What is CodeSphere?

CodeSphere is a structured coding ecosystem designed to enhance learning, evaluation, and examination processes. It provides an integrated environment where students can write and execute code, teachers can evaluate submissions, and administrators can manage system operations seamlessly.

For Students

  • View the algorithm provided by the teacher.
  • Write and execute code inside the built-in IDE.
  • Debug and refine solutions using the terminal.
  • Participate in viva sessions conducted by the chatbot.
  • Track performance and improve coding efficiency.

For Teachers

  • Upload or modify algorithms for coding sessions.
  • Analyze student submissions and viva responses.
  • Assign manual marks where necessary.
  • Monitor active students and performance metrics.
  • Enable or disable exam mode when required.

For Administrators

  • Manage student and teacher accounts.
  • Control system-wide examination settings.
  • Assign or remove users.
  • Maintain overall system integrity and operations.
